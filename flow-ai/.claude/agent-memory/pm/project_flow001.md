PRD v2.3 작성 완료 (2026-04-15, 상태: draft). 저장 경로: `docs/pm/prd/FLOW-001-ai-smart-secretary-prd.md`

Phase 1 FRD v1.0 작성 완료 (2026-04-15, 상태: 초안). 저장 경로: `docs/pm/specs/FLOW-001-phase1-spec.md`

**Why:** v2.3에서 캐릭터(플로키/펭귀니/거북이) 기획 전면 제거. 통일 AI 아바타 1종(보라색 구체 + 다크 바이저 + 흰 눈) 도입, 온보딩 5단계 → 4단계 축소, 대화 스타일 7종 → 브리핑 스타일 4종(요약형/분석형/친근형/지시형)으로 변경.

**How to apply:** FLOW-001 관련 후속 문서(기능 스펙, 핸드오프, 의사결정 로그) 작성 시 이 PRD v2.3을 기준 문서로 참조. 캐릭터 관련 내용은 일절 제외.

핵심 결정사항 (v2.3 기준):
- AI 아바타: 통일 1종 (보라색 구체 + 다크 바이저 + 흰 눈), 선택 기능 없음
- 온보딩: 4단계 — Step 1(브리핑 스타일) → Step 2(오전 브리핑) → Step 3(일정 준비) → Step 4(퇴근 리포트)
- 브리핑 스타일 4종: 요약형(summary) / 분석형(analysis) / 친근형(friendly) / 지시형(directive), 기본값: 요약형
- AI 응답 헤더: 아바타 + "AI 비서" + 시간 (캐릭터 이름 없음)
- 홈 프로필 카드: 아바타 + 인사말 (캐릭터 이름 제거)
- localStorage: `flowai_character` 제거, `flowai_style` + `flowai_customized` 유지
- 설정 화면: AI 캐릭터 섹션 제거, 브리핑 스타일 섹션만 유지
- 퇴근 마무리 리포트: 18:00 이후 마지막 접속 1시간 뒤 행동 기반 트리거 (유지)
- 오전 브리핑 데이터 수집: 알림 시각 30분 전부터 수집 시작 (유지)
- 오픈 이슈 9건 (OI-001~OI-009) — OI-010(캐릭터 이름 고정 여부) 삭제됨
- 정식 출시 목표: 2026-07-15 (유지)
