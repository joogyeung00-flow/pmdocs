# FLOW-001 AI 스마트 비서 — 화면설계서

---

## 1. 문서 개요

본 문서는 AI 스마트 비서의 전체 화면 흐름과 각 화면의 UI 스펙을 정의합니다.
캐릭터 3종(플로키/펭귀니/거북이), 대화 스타일 7종, 알림 카드 기반 홈, AI 퀵 액션 패턴을 기준으로 설계되었습니다.
**Flow AI 디자인 시스템**(`design-system/design-tokens/`)의 토큰을 기준으로 설계되었습니다.

### 적용 디자인 토큰 요약

| 토큰 | 값 | 소스 |
|------|----|------|
| 브랜드 컬러 | `brand-500` = `#5B40F8` | colors.json |
| 브랜드 라이트 | `brand-50` = `#F3F1FF` | colors.json |
| 브랜드 보더 | `brand-100` = `#E9E5FF` | colors.json |
| 기본 폰트 | Pretendard | typography.json |
| 영문 폰트 | Inter | typography.json |
| 기본 단위 | 4px | spacing.json (baseUnit: 4) |
| 카드 패딩 | 24px, radius 8px | spacing.json (componentDefaults.card) |
| 채팅 버블 | px:12, py:8, radius:6 | spacing.json (componentDefaults.chatBubble) |
| 버튼 | px:16, py:8, radius:6 | spacing.json (componentDefaults.button) |
| 배지 | px:8, py:4, radius:full | spacing.json (componentDefaults.badge) |
| 애니메이션 | fadeIn 150ms, slideUp 250ms spring | motion.json |

## 2. 진입 구조

### 진입 경로

| 진입 경로 | 트리거 | 도착 화면 | 조건 |
|----------|--------|----------|------|
| 사이드바 메뉴 | "AI 비서" 탭 | scr-onboard | 최초 진입 (온보딩 미완료) |
| 사이드바 메뉴 | "AI 비서" 탭 | scr-100 (AI 비서 홈) | 온보딩 완료 후 |
| Push 알림 | 오전 브리핑 알림 탭 | scr-110-briefing | - |
| Push 알림 | 일정 리마인더 알림 탭 | scr-110-reminder | - |
| Push 알림 | 퇴근 마무리 알림 탭 | scr-110-report | - |
| 알림 카드 탭 | scr-100 내 알림 카드 "확인하기" | scr-110 (해당 세션) | - |

### 전체 플로우

```
[사이드바 "AI 비서"]
    │
    ├── 온보딩 미완료
    │   └──▶ [scr-onboard] Step 1 캐릭터 선택
    │              └──▶ Step 2 대화 스타일 선택
    │                        └──▶ "시작하기" → [scr-100]
    │
    └── 온보딩 완료
        └──▶ [scr-100 AI 비서 홈]
                  ├── 알림 카드 "확인하기" → [scr-110 해당 세션]
                  ├── 지난 대화 항목 탭 → [scr-110 해당 세션]
                  └── ⚙️ 탭 → [scr-settings]
```

### 초기값 및 localStorage

| 항목 | 초기값 | 변경 방법 |
|------|--------|----------|
| 캐릭터 | 미설정 (온보딩 필수) | scr-onboard 또는 scr-settings |
| 대화 스타일 | 미설정 (온보딩 필수) | scr-onboard 또는 scr-settings |
| 온보딩 완료 여부 | false | 온보딩 완료 시 true |

## 3. 온보딩 (scr-onboard)

### 화면 목적

최초 AI 비서 진입 시 캐릭터와 대화 스타일을 선택하는 2단계 설정 화면.
온보딩 완료 후 다시 진입하지 않음 (localStorage `flowai_customized` = true).

### 공통 레이아웃 구조

```
┌─────────────────────────────────────┐
│  Header (px:16, py:12)              │
│  ✕                    AI 맞춤 설정  │
├─────────────────────────────────────┤
│  스텝 인디케이터 (mt:8, 중앙정렬)   │
│     ● ○    (dot: 8×8px, gap:6px)   │
│  (활성: brand-500, 비활성: gray-200) │
├─────────────────────────────────────┤
│  콘텐츠 영역 (스크롤, flex-1)       │
│  [Step별 내용]                      │
├─────────────────────────────────────┤
│  하단 버튼 영역 (px:24, py:20)      │
│  [주 버튼]                          │
│  [스킵/보조 버튼] (선택적)          │
└─────────────────────────────────────┘
```

**스텝 인디케이터 스펙**
- dot 크기: w-8px h-1.5px (활성), w-8px h-1.5px (비활성)
- 형태: rounded-full
- 활성 색: `brand-500`
- 비활성 색: `gray-200`
- gap: 6px

**캐릭터 카드 상태**

| 상태 | border | background | 설명 |
|------|--------|-----------|------|
| default | gray-100 | white | 미선택 |
| selected | brand-500 | brand-50 | 선택됨 (border 2px) |
| hover | gray-200 | gray-50 | hover |

**"인기" 태그 (플로키 전용)**
- 위치: 카드 우상단 (absolute, top:8, right:8)
- 스타일: bg-brand-500, text-white, text-xs, px:6, py:2, rounded-full

**캐릭터 목록**

| 캐릭터명 | 이모지 | 설명 | 특이사항 |
|---------|--------|------|---------|
| 플로키 | 🐙 | 호기심 많은 바다 탐험가 | "인기" 태그 |
| 펭귀니 | 🐧 | 남극에서 온 업무 도우미 | - |
| 거북이 | 🐢 | 믿음직한 숲속 길잡이 | - |

**"다음" 버튼 상태**
- 미선택: `bg-gray-200`, `text-gray-400`, `cursor-not-allowed`, disabled
- 선택 후: `bg-brand-500`, `text-white`, 활성화

| 스타일명 | 설명 |
|---------|------|

| 진중한 | 격식 있고 신중한 |
| 친근한 | 친구처럼 가깝고 따뜻한 |
| 솔직한 | 직접적이고 명확한 |
| 독특한 | 창의적이고 재미있는 |
| 간결한 | 핵심만 짧고 명료하게 |
| 까칠한 | 날카롭고 거침없는 |

**"시작하기" 버튼 동작**
1. localStorage에 캐릭터, 대화 스타일, `flowai_customized: true` 저장
2. scr-100 (AI 비서 홈)으로 화면 전환 (slideUp 250ms spring)

**"캐릭터 다시 선택"**
- 탭 시 Step 1로 복귀 (캐릭터 선택 초기화 없음, 현재 선택 유지)

**스킵 처리 ("나중에 설정할게요")**
- Step 1에서 스킵 탭 시: 캐릭터 플로키(기본), 대화 스타일 기본적인으로 자동 설정 후 scr-100 진입
- `flowai_customized: true` 저장 (재진입 방지)

## 4. AI 비서 홈 (scr-100)
